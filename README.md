# Proyecto: funcionesJava  
_Ejercicios prácticos de funciones y modularización en Java_

## Descripción
Este proyecto contiene una serie de ejercicios en **Java** que aplican la creación y uso de **funciones (métodos)** para modularizar el código, dividir responsabilidades y mejorar la reutilización.  
Forma parte de un portafolio académico y profesional orientado a la práctica de **programación estructurada y modular**.

---

## Objetivos
- Comprender la definición y uso de funciones en Java.  
- Modularizar programas dividiendo la lógica en métodos auxiliares.  
- Desarrollar programas con **código limpio** y bien documentado.  
- Practicar paso de parámetros y retorno de valores.  
- Aplicar funciones en problemas prácticos de lógica y algoritmos.  

---

## Contenido del proyecto

| Ejercicio             | Clase                     | Descripción |
|-----------------------|---------------------------|-------------|
| **calcularFactorial** | `calcularFactorial.java`  | Obtiene el factorial de un número usando una función auxiliar. |
| **calcularPotencia**  | `calcularPotencia.java`   | Calcula la potencia de un número con base y exponente mediante funciones. |
| **calcularPromedio**  | `calcularPromedio.java`   | Calcula el promedio de un conjunto de valores ingresados por el usuario. |

---

## Contenido temático
- **Funciones básicas** → definición y llamada de métodos.  
- **Parámetros y retorno** → paso de valores y resultados.  
- **Funciones auxiliares** → dividir la lógica en métodos pequeños.  
- **Reutilización de código** → aplicar funciones en distintos contextos.  
- **Aplicaciones prácticas** → cálculos matemáticos y promedios.
